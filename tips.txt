# Criar novo projeto SvelteKit com nome "app"
npx sv create app

## responder:
tipo projeto: sveltekit minimal
typescript checking: no (para simplicidade do exemplo)
selecionar: prettier e eslint
package manager: npm

## para rodar o projeto modo dev
cd app
npm run dev
npm run dev -- --open

# para instalar uma "fake rest API":
npm i json-server
## criar arquivo db.json no mesmo dir que "packaje.json"
## para iniciar a rest API:
npm run server-json
